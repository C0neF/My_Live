# 布局改动详解（A / B 共用）

> 范围:**完整重设计**。本文件只覆盖「会真正改变结构/布局」的三处,其余改动是纯配色/视觉(见 `plan-a-dark-first.md` / `plan-b-follow-system.md` 第 1 节与 2.4–2.6)。
> 房间页（播放器 + 聊天布局）**不在改动范围**,仅调可读性与颜色。
> 仍为设计规格,不直接改 app 代码。

布局改动只有三处:① 房间卡 ② 首页平台选择器 ③ 我的页。

---

## ① LiveRoomCard（`component/LiveRoomCard.kt`）

### 现在
```
┌───────────────────────────┐  ← Card：1px 描边 + surfaceVariant·25% 填色，圆角16
│ ┌───────────────────────┐ │
│ │[平台▮渐变]         封面 │ │  ← 16:9 封面
│ │              ● 人气     │ │
│ └───────────────────────┘ │
│ (◎头像36·描边环) 标题…1行  │  ← 头像带「premium border」环
│              👤 主播名      │     标题被头像挤成单行
└───────────────────────────┘
```

### 改后
```
┌───────────────────────┐      ← 仅封面：圆角12，无卡容器 / 无描边 / 无填色
│[平台]             封面  │
│              ● LIVE 人气│
└───────────────────────┘
标题第一行 占满宽，可两行          ← 8dp 间距后，裸 bg 上；ink·SemiBold·maxLines 2
◎ 主播名                         ← 18–20dp 小头像（无环）+ muted 名
```

### 变化点
- 去掉 `Card` 容器、1px 描边、填色（`:51-54`）→ 封面自己即边界。
- 平台徽章:`horizontalGradient` → 平台品牌**纯色**小 chip（`:98-106`）。
- 删头像「premium border」环（`:157`）。
- **信息块重排（核心布局变化）**:标题升为主行、占满宽、可 2 行;头像缩到 18–20dp 移到次行与主播名同排。标题不再被头像挤压。
- 卡片整体高度基本持平(封面同比例,信息区两行),网格列数/间距不变。

## ② 首页平台选择器（`screen/home/HomeScreen.kt:184-221`）

### 现在
```
[ 推荐 ][ 斗鱼 ][ 虎牙 ][哔哩哔哩]   ← 4 个等宽 weight(1f) 药丸 chip
                                     选中项整块填平台色
```

### 改后
```
 推荐    斗鱼    虎牙    哔哩哔哩       ← 可横向滚动 tab 行（文字按内容宽）
 ━━━━                                ← 选中项下方 2dp 平台品牌色下划线
```

### 变化点
- 从「等宽药丸组」改为「可滚动 tab 行 + 下划线指示」。
- 平台多于 4 个时可横滑,不再被等分挤窄。
- 选中态语义更清楚:「我正在逛斗鱼」靠下划线 + 品牌色表达,而非整块填色。
- 备选形态:Material 分段控件(SegmentedButton);若想最贴原生 Android,优先 tab 行。

## ③ 我的页（`screen/mine/MineScreen.kt`）

### 现在
```
我的
┌──────────────────────────┐   ← profile 英雄卡：圆角24 + 描边 + 填色
│ (M)  MyLive 用户           │
│      极致、纯粹的直播体验    │   ← 营销文案；且无真实账户
└──────────────────────────┘
┌──────────────────────────┐   ← 卡 1（独立描边卡）
│ 历史记录              ›    │
└──────────────────────────┘
┌──────────────────────────┐   ← 卡 2
│ 账号管理              ›    │
└──────────────────────────┘
        … 共 5 张相同描边卡 …
```

### 改后
```
我的

  历史记录                    ›
  ────────────────────────       ← hairline 行间分隔
  账号管理                    ›
  ────────────────────────
  同步                        ›
  ────────────────────────
  工具                        ›
  ────────────────────────
  设置                        ›

  MyLive · v1.x.x                 ← 版本号弱化收底
```

### 变化点
- **5 张独立描边卡 → 一组分组列表(核心布局变化)**:行内嵌图标 + 标题 + chevron,行间用 `hairline` 分隔,复用 `SettingsMenu` 行样式。
- profile 英雄卡:无真实账户,**整块移除**直接进列表(或缩成一行安静标题)。
- 删营销文案「极致、纯粹的直播体验」。
- 去掉 5 圈描边 + 5 块填色,信息密度更高、更原生、更好扫视。
